# ArrayHof 예제

## Mission

- 자바스크립트 배열관련 메서드를 이용해 데이터 조작 연습

### 주의사항

- for문을 사용 금지
- 부수 효과 최소화
- 데이터의 변화에 따른 이벤트 감지가 용이한 코드
- 가독성을 고려한 메서드, 매개변수 네이밍
- 가독성에 좋은 인덴트 주의
- ESLint 적용

### 주요 메서드 및 문법

- Array.prototype.map
- Array.prototype.forEach
- Array.prototype.reduce
- Array.prototype.sort
- Arrow function
- Spread operator
